# desktop_task_list

A modern and simple task tracking application. 

# Installation & Usage
1. Download Source Code as a .ZIP file.
2. Extract Files from .ZIP
3. Open directory in _Terminal_ or _PowerShell_
4. Run the following:
```
dotnet build
dotnet run
```
